# TF_IDF_based_MapReduce(基于云和MapReduce的文本检索)
## 要求
基于Hadoop MapReduce的简易站内搜索引擎。
爬虫爬取某个网站的数据、对文本做标准化（拆分标点、转小写、分词等），实现基于MapReduce的倒排索引（TF-IDF），和简单的网页接口。用Docker进行部署。
软件栈：Docker、Hadoop（可使用DockerHub的Hadoop镜像）
